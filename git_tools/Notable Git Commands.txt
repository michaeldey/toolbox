Notable Git Commands:

	Clone a remote repository:
		git clone <repository name>
		Example:
			git clone https://github.com/michaeldey/toolbox.git


	Create a new branch:
		git branch <branch name>
		Example:
			git branch dev


	Checkout a branch:
		git checkout <branch name>
		Example:
			git checkout dev


	Push and connect the branch to the remote repository
		git push --set-upstream origin <branch name>
		Example:
			git push --set-upstream origin dev


	Add a file to local staging environment
		git add <file name>
		Example:
			git add Notable_Git_Commands.txt


	Add all changed files to local staging environment
		git add -A


	Make a commit to the local staging environment
		git commit -m "this is my commit message"


	push a commit to the remote repository
		git push

	delete a file and have GIT see it
		git rm <filename>

	see status of current git branch
		git status

	see git log
		git log
	compact version
		git log --oneline
	most recent commit
		git log -n 1
	most recent commit on a specific branch
		git log -n 1 <branch name>
	most recent commit hash
		git log -n 1 --pretty=format:"%H"
		

	merge branch into another branch
		checkout the branch you want to merge into
		merge the branch
		push the branch to remote repo
		Example:
			git checkout master
			git merge dev
			git push
	
	revert
		git revert <hash>

	see most recent commit hash:
		git log -n1 --format=format:"%H"


	M2 Git Commands (in order):
		git checkout integration						# from gitlab
		git pull
		git merge -no-ff feature/whatever
		git push 										# to gitlab
		git checkout staging
		git pull
		git merge --no-ff integration
		git push origin staging 						# to gitlab (origin remote)
		git push cloud staging 							# to Magento cloud remote
